How to connect the LEDs and the Button:

LEDs:
----------------------------------------------------------------------------------------------------
Red     = Pin 2
Blue    = Pin 3
Green   = Pin 4
Yellow  = Pin 5

Buttons:
----------------------------------------------------------------------------------------------------
Start/Reset Button: Pin 9
Button 1:           Pin 10
Button 2:           Pin 11
Button 3:           Pin 12
Button 4:           Pin 13
